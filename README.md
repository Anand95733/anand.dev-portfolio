# Anand Ediga - Portfolio Website

A modern, responsive portfolio website built with React, Vite, and Tailwind CSS showcasing my work as a Full Stack Developer.

## About

This portfolio showcases my experience in full-stack development, including:
- 8+ months of industry experience
- Expertise in React.js, Next.js, Angular, and FastAPI
- AI/ML integration and cloud deployment
- Real-time collaborative applications
- Production-grade web applications

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **Build Tool**: Vite
- **State Management**: Redux Toolkit
- **Form Handling**: Formik
- **Email Service**: EmailJS
- **Icons**: React Icons
- **Smooth Scrolling**: React Scroll

## Features

- Responsive design for all devices
- Smooth animations with Framer Motion
- Interactive project showcase
- Skills visualization
- Experience timeline
- Contact form with email integration
- Dark theme with custom color palette

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
